# Custos - Installation & Setup

Thank you for installing **Custos**. This document outlines how your environment has been configured.

## 🛠️ Automated Setup
This installer has performed a check for the necessary dependencies:

*   **Node.js:** If not detected, the installer has automatically deployed the required version.
*   **Python:** If not detected, the installer has automatically deployed the required version and added it to your system PATH.

*If you already had these installed, Custos has been configured to use your existing local versions.*

## 🚀 How to Use Custos
Once installation is complete, you can interact with the application using the following methods:

*   **Starting/Stopping:** Use the shortcuts created on your **Desktop** or in your **Start Menu** to launch or terminate the Custos service.
*   **Configuration:** Need to update your environment variables?
    1. Right-click any folder on your computer.
    2. Select **Custos** from the context menu.
    3. Click **Edit ERP.env** to open your configuration file in Notepad.

## 🛡️ Support
If you encounter any issues during operation:
1. Ensure you have the necessary read/write permissions in the `C:\Custos` directory.
2. If the service fails to start, verify your `erp.env` configuration is valid.
